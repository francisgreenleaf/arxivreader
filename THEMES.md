# ArXiv Reader - Theme Documentation

This document provides comprehensive information about all available themes in the ArXiv Reader application.

## Overview

The ArXiv Reader now features **10 distinct themes**, each designed with specific use cases and aesthetic preferences in mind. All themes are fully responsive and optimized for reading academic papers.

## Technical Implementation

### Theme Structure

Each theme consists of:
1. **CSS File:** Located in `src/public/css/themes/[theme-id].css`
2. **Configuration:** Defined in `src/public/js/theme-manager.js`
3. **CSS Variables:** Custom properties for colors, spacing, etc.

### CSS Variables

All themes use CSS custom properties (variables) for consistency:

```css
:root.theme-[id] {
    /* Color Variables */
    --primary-color: ...;
    --secondary-color: ...;
    --success-color: ...;
    --danger-color: ...;
    --warning-color: ...;
    --info-color: ...;
    
    /* Background Colors */
    --bs-body-bg: ...;
    --bs-body-color: ...;
    
    /* Surface Colors */
    --surface-color: ...;
    --surface-hover: ...;
    --border-color: ...;
    --text-muted: ...;
    
    /* Paper Content */
    --paper-bg: ...;
    --paper-text: ...;
    --paper-heading: ...;
    --paper-abstract-bg: ...;
    
    /* Interactive Elements */
    --btn-bg: ...;
    --btn-border: ...;
    --btn-hover-bg: ...;
    
    /* Card Elements */
    --card-bg: ...;
    --card-border: ...;
    --card-hover-shadow: ...;
}
```

## Switching Themes

Themes can be switched through:
1. **Theme Selector** in the UI
2. **Keyboard Shortcuts** (if implemented)
3. **Programmatically** via `ThemeManager.loadTheme(themeId)`

The selected theme is automatically saved to localStorage and persists across sessions.

## Accessibility Considerations

### Contrast Ratios

All themes except Cyberpunk aim for WCAG AA compliance minimum:
- **Normal text:** 4.5:1 contrast ratio minimum
- **Large text:** 3:1 contrast ratio minimum
- **High Contrast theme:** AAA compliance (7:1 for normal text)

### Focus Indicators

- All themes provide visible focus indicators
- High Contrast theme has enhanced 3px focus rings
- Keyboard navigation fully supported

### Text Sizing

- All themes respect font size multiplier setting
- High Contrast theme has larger default sizes
- Relative units used for scalability

## Contributing

To contribute a new theme:

1. Create a new CSS file in `src/public/css/themes/`
2. Follow the existing theme structure
3. Define all required CSS variables
4. Style all component selectors
5. Add theme configuration to `theme-manager.js`
6. Test across all major components
7. Verify accessibility (WCAG AA minimum)
8. Submit pull request with theme documentation
